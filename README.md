# Defax

## Git

Los archivos de pesos de modelos con extensión .pth se ignoran automáticamente por Git en cualquier subcarpeta del proyecto mediante la regla definida en [.gitignore](.gitignore).

> Nota: esta regla también aplica a los documentos Markdown del repositorio para mantener claro que los archivos de pesos no deben enviarse al control de versiones.
